
# **The Experiments with IR require the following dependencies**

## **Python Packages:**
Install IRCODEC and PIGPIO packages

```bash
pip install pigpio
pip install ircodec
```

## **pigpio Daemon:**
Download and Make Pigpio daemon
```bash
wget https://github.com/joan2937/pigpio/archive/master.zip
unzip master.zip
cd pigpio-master
make
sudo make install
```

## **Start pigpio daemon**
```bash
sudo pigpio
```
