
# **The Experiments with IR require the following dependencies**

## **Python Packages:**
-  pip install pigpio
-  pip install ircodec

## **Pigpio Daemon:**

- > wget https://github.com/joan2937/pigpio/archive/master.zip
  >
  > unzip master.zip
  >
  > cd pigpio-master
  >
  > make
  >
  > sudo make install
